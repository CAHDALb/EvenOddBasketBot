EvenOddBasketBot — установка версии с кнопочным меню
============================================================

Что добавлено:
- /profile и кнопка «👤 Мой профиль»;
- /menu и постоянное главное меню;
- «📈 Моя статистика»;
- «💎 Купить Premium»;
- «ℹ️ Помощь»;
- отдельная кнопочная «👑 Админ-панель»;
- кнопки управления пользователями;
- защита от повторного повышения бывшего администратора после рестарта.

ВАЖНО ПРО СЕКРЕТЫ
-----------------
Файла .env в архиве нет.
Свой локальный .env не удаляйте и не загружайте на GitHub.
Переменные BOT_TOKEN и DATABASE_URL на Render не меняются.

ВАЖНО ПРО ВТОРОГО АДМИНИСТРАТОРА
--------------------------------
Откройте Render:
Service -> Environment

Проверьте переменные:
- ADMIN_TELEGRAM_IDS
- CHAT_IDS

Оставьте в административном списке только свой Telegram ID.
ID друга удалите. После изменения сохраните настройки.

КАК ЗАГРУЗИТЬ
-------------
1. Распакуйте архив.
2. Скопируйте файлы проекта в папку своего репозитория.
3. Не копируйте и не удаляйте свой .env.
4. Выполните:

   git add .
   git commit -m "Add Telegram profile and button menu"
   git push

5. Дождитесь успешного Deploy на Render.
6. Отправьте боту:

   /start

или:

   /menu

--------

- 👤 Мой профиль

Администратор дополнительно увидит:
- 👑 Админ-панель

В админ-панели проверьте:

- ✅ Разблокировать
