============================================================
EvenOddBasketBot — админская рассылка
============================================================

Что добавлено:
- кнопка «📢 Рассылка» в админ-панели;
- публикации из текста, фото, видео, анимации, документов и аудио;
- обычные Telegram-опросы через штатный конструктор;
- предпросмотр перед отправкой;
- выбор аудитории: все / PREMIUM / FREE;
- обычная отправка или отправка с закреплением;
- финальное подтверждение;
- итоговый отчёт: доставлено, ошибки, закреплено;
- рассылка работает в отдельном потоке и не останавливает проверку матчей.

ВАЖНО
------
Файла .env в архиве нет. Ваш токен и DATABASE_URL остаются у вас.
Новых переменных окружения добавлять не требуется.

КАК ОБНОВИТЬ НА RENDER
----------------------
1. Распакуйте архив.
2. Скопируйте содержимое папки проекта в свой репозиторий.
3. Не удаляйте свой локальный .env.
4. Выполните:

   git add .
   git commit -m "Add admin broadcast wizard"
   git push

5. Дождитесь успешного Deploy на Render.
6. В Telegram откройте:

   👑 Админ-панель -> 📢 Рассылка

КАК ПОЛЬЗОВАТЬСЯ
----------------
1. Нажмите «📝 Создать публикацию» или «📊 Создать опрос».
2. Отправьте готовый пост либо создайте обычный Telegram-опрос.
3. Проверьте предпросмотр.
4. Выберите аудиторию:
   - 📢 Отправить всем;
   - 💎 Только PREMIUM;
   - 🆓 Только FREE.
5. Выберите:
   - 📤 Отправить;
   - 📌 Отправить и закрепить.
6. Проверьте итоговую карточку и нажмите «✅ Подтвердить».

До финального подтверждения пользователи ничего не получат.

ОГРАНИЧЕНИЯ ЭТОЙ ВЕРСИИ
-----------------------
- за одну рассылку отправляется одно Telegram-сообщение;
- альбомы из нескольких фото/видео пока не объединяются;
- черновик хранится в памяти и исчезает при перезапуске Render;
- каждый получатель получает отдельную копию обычного опроса.
