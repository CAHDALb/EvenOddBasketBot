===========================================================
EvenOddBasketBot 14.0 — фирменное оформление
===========================================================

ЧТО ДОБАВЛЕНО

1. Фирменный приветственный баннер при /start.
2. Кнопка «🏀 О проекте» и команда /about.
3. Полный дисклеймер и правила ответственного использования.
4. Единый стиль профиля, меню, Premium, админ-панели и сигналов.
5. Автоматическая настройка имени, описания и команд Telegram-бота.
6. Новый дизайн Mission Control на сайте Render.
7. Фирменный favicon и изображения проекта.

ВАЖНО

Файла .env в архиве нет. Ваш токен и DATABASE_URL остаются только у вас.
Новые обязательные переменные окружения не нужны.

КАК ОБНОВИТЬ

1. Распакуйте архив.
2. Скопируйте файлы в папку проекта с заменой.
3. Убедитесь, что ваш локальный .env остался на месте.
4. Выполните:

   git add .
   git commit -m "Add EvenOddBasketBot brand identity v14"
   git push

5. Дождитесь успешного Deploy на Render.
6. Отправьте боту /start.
7. Откройте адрес Render и проверьте новый Mission Control.

АВАТАР

Telegram Bot API не меняет фотографию профиля бота автоматически.
Установите файл assets/brand/avatar_eob.png вручную через @BotFather.
Подробные шаги находятся в BOTFATHER_SETUP.txt.
