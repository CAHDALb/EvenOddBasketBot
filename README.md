# EvenOddBasketBot

**Basketball Signal Analytics**

EvenOddBasketBot — Telegram-сервис алгоритмической аналитики LIVE-баскетбольных матчей с PostgreSQL, системой FREE/PREMIUM/ADMIN, прозрачной статистикой и административной рассылкой.

## Версия 14.0

- фирменный Telegram-интерфейс;
- логотип, аватар и приветственный баннер;
- профиль пользователя и тарифы;
- админ-панель и массовые публикации;
- Mission Control в едином спортивно-технологическом стиле;
- дисклеймер ответственного использования.

Инструкция по обновлению: `README_INSTALL_BRANDING.txt`.
