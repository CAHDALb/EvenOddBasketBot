EvenOddBasketBot 15.2 — небольшие исправления интерфейса

Изменения:
1. Увеличен шрифт карточки последнего сигнала.
2. Удалён блок DATA ENGINE / ACTIVE из бокового меню.
3. Убрана лишняя вертикальная прокрутка бокового меню на компьютере.
4. Индикатор LIVE теперь зелёный, в одном стиле с WIN.

Установка:
- Скопируйте файлы проекта с заменой.
- Свой .env оставьте без изменений.
- Выполните:
  git add .
  git commit -m "Polish dashboard v15.2"
  git push
- После Deploy откройте сайт с Ctrl+F5.
