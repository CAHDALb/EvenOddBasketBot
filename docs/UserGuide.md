# Пользовательское меню Telegram

## Главное меню

После команды `/start` или `/menu` бот показывает постоянные кнопки:

- **👤 Мой профиль** — имя, Telegram ID, тариф, статус и лимит;
- **📈 Моя статистика** — количество полученных FREE-сигналов за текущий день;
- **💎 Купить Premium** — описание различий FREE и PREMIUM;
- **ℹ️ Помощь** — список доступных команд;
- **👑 Админ-панель** — отображается только пользователю с тарифом `admin`.

## Текстовые команды

```text
/start
/menu
/profile
/status
/myid
/help
```

Кнопки и текстовые команды работают параллельно.

## Административное меню

В административной панели доступны кнопки:

- просмотр пользователей;
- выдача Premium;
- перевод в FREE;
- блокировка;
- разблокировка;
- возврат в главное меню.

Для действий, которым нужны Telegram ID или срок подписки, бот показывает готовый формат команды.
