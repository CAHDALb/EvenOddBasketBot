# EvenOddBasketBot Developer Book

## Глава 13. Административная панель

### Цель этапа

Управлять пользователями прямо из Telegram, не открывая Neon и не изменяя код вручную.

### Архитектура

```text
Telegram-команда администратора
            ↓
      telegram_bot.py
            ↓
      admin_commands.py
            ↓
      telegram_users.py
            ↓
       PostgreSQL Neon
```

### Добавленные команды

- `/users` — общая статистика пользователей;
- `/premium TELEGRAM_ID DAYS` — выдача Premium;
- `/free TELEGRAM_ID` — перевод на FREE;
- `/block TELEGRAM_ID` — блокировка;
- `/unblock TELEGRAM_ID` — снятие блокировки.

### Защита

Каждая административная команда проверяет тариф отправителя. Выполнение разрешено только пользователю с тарифом `admin`.

### Важное решение

Административная логика вынесена в отдельный файл `admin_commands.py`. Это не даёт `telegram_bot.py` разрастись и позволяет позже добавить рассылки, платежи и отчёты без смешивания разных частей проекта.

### Версия

Этап формирует версию **v0.7 — Admin Panel**.
