# Административная панель

## Просмотр статистики

```text
/users
```

## Выдача Premium

```text
/premium 123456789 30
```

Где `123456789` — Telegram ID пользователя, а `30` — срок в днях.

## Перевод на FREE

```text
/free 123456789
```

## Блокировка

```text
/block 123456789
```

## Разблокировка

```text
/unblock 123456789
```

Пользователь должен хотя бы один раз отправить боту `/start`, чтобы появиться в базе.
