# Административная панель

Администратор может открыть панель кнопкой **👑 Админ-панель**. Эта кнопка не отображается пользователям FREE и PREMIUM.

## Просмотр пользователей

Кнопка **👥 Пользователи** или команда:

```text
/users
```

Следующая страница списка:

```text
/users 2
```

## Выдача Premium

Кнопка **💎 Выдать Premium** покажет подсказку. Команда:

```text
/premium 123456789 30
```

Где `123456789` — Telegram ID пользователя, а `30` — срок в днях.

## Перевод на FREE

```text
/free 123456789
```

Администратор может перевести другого администратора в FREE, но не может снять права у самого себя.

## Блокировка

```text
/block 123456789
```

## Разблокировка

```text
/unblock 123456789
```

Пользователь должен хотя бы один раз отправить боту `/start`, чтобы появиться в базе.

## Важно про ADMIN_TELEGRAM_IDS

Переменная Render `ADMIN_TELEGRAM_IDS` нужна для первоначального создания владельца бота. В ней рекомендуется оставить только Telegram ID владельца.

При каждом запуске код больше не повышает уже существующего пользователя до ADMIN повторно. Поэтому перевод пользователя командой `/free` сохраняется после перезапуска Render.
