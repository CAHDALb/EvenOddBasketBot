# EvenOddBasketBot Brand Guide 14.0

## Позиционирование

EvenOddBasketBot — серьёзный спортивно-технологический сервис аналитики LIVE-баскетбольных матчей. Коммуникация строится вокруг прозрачной статистики, алгоритмического отбора и ответственного отношения к риску.

## Название и слоган

- Название: **EvenOddBasketBot**
- Слоган: **Basketball Signal Analytics**
- Короткое обозначение: **EOB**

## Цветовая система

- Deep Navy: `#031126`
- Dark Background: `#020817`
- Analytics Cyan: `#19D5DF`
- Basketball Orange: `#FF7A00`
- Text White: `#F5FBFF`
- Secondary Text: `#9BB0C5`

Бирюзовый обозначает аналитику, данные и системность. Оранжевый обозначает баскетбол, LIVE-события и важные акценты.

## Стиль сообщений

Сообщения начинаются с единого заголовка вида:

```text
🏀 EVENODDBASKETBOT • LIVE-СИГНАЛ
━━━━━━━━━━━━━━━━━━━━
```

Текст должен быть спокойным, точным и без обещаний гарантированного заработка.
