EvenOddBasketBot 15.1 — обновление сайта
========================================

Что изменено:
- в краткой истории убрана колонка EVEN/ODD;
- исправлено наложение логотипа на заголовки разделов;
- вкладка «Банк» показывает три реальные модели 0.25%, 0.5% и 1%;
- кнопка «Настроить банк» пересчитывает модели для суммы пользователя;
- под Strategy Monitor показывается последний сигнал и статистика его страны/лиги;
- увеличен шрифт журнала сигналов;
- добавлены кнопки перехода в Telegram-бота.

Render Environment:
TELEGRAM_BOT_USERNAME=имя_бота_без_символа_@
DASHBOARD_DEFAULT_BANKROLL=10000

TELEGRAM_BOT_USERNAME рекомендуется добавить, чтобы ссылка была доступна
сразу. Если переменная отсутствует, сайт попробует безопасно определить
username через Telegram getMe по BOT_TOKEN.

Установка:
git add .
git commit -m "Update live dashboard to v15.1"
git push
